Activities

Activity 01

Installing and Comparing an Operating System in VirtualBox

This activity focused on setting up a virtualized computing environment. We installed a Linux operating system (OS) within VirtualBox, which involved creating and configuring a virtual machine (VM). This setup provides a safe, isolated environment for practicing system administration and experimentation without affecting the host operating system.

Activity 02

Alison Course Completion + Helpdesk Support Simulation

This activity centered on developing essential practical system administration skills. Students were tasked with managing key operational aspects of a Linux system (specifically Ubuntu running on VirtualBox), which included managing user accounts, setting and understanding file permissions, controlling system services, and implementing basic security measures. The primary goal was to reinforce hands-on OS administration proficiency and a practical understanding of platform security.

Activity 03

Cross-Platform OS Administration and Security

The project portfolio is a comprehensive task requiring students to demonstrate web development and deployment skills.
